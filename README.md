# ray-skills

这个仓库保存我使用 agent 过程中总结生成的个人使用的 skills。

## 目录结构

```
skills/
├── coding/          # 编程相关技巧
├── prompting/       # 提示词技巧
├── tools/           # 工具使用技巧
└── workflows/       # 工作流技巧
```

## 使用说明

每个 skill 文件记录以下信息：

- **场景**：适用的使用场景
- **技巧**：具体的操作步骤或方法
- **示例**：实际使用示例
- **注意事项**：使用时需要注意的点
